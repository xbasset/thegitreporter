API Discovery: API Sirene Turns Company Identity Into Agent Checks

INSEE's API Sirene open data gives agents a public way to check French company and establishment identity, with SIREN/SIRET scope and diffusion caveats close to the source.

API Sirene lets builders query the Sirene register for legal units and establishments, including current and historical data, with account access and a 30-requests-per-minute open-data rate limit listed in the data.gouv.fr API record.

Paste This Into Your AI Agent

Use API Sirene open data only for company identity checks.
Given a French SIREN or SIRET, verify:
1. the identifier exists in the INSEE Sirene source;
2. the legal unit or establishment status matches the user's claim;
3. dateDernierTraitement is recent enough for the decision;
4. diffusion rules allow the data to be displayed.
Return: identifier checked, source URL, matched fields, caveat, and what to re-check next.

What To Watch

Operators can use the API as a grounding step before an agent drafts procurement, grant, vendor, or onboarding checks. The caveat is practical and legal: registration, quotas, terms of use, and partial-diffusion rules matter before any agent republishes company data.

API Discovery: API Adresse Turns French Addresses Into Agent Checks

API.gouv's API Adresse page exposes a fully open geocoding source that agents can use to validate French addresses before routing, mapping, or form decisions.

Open Edition Evidence below

API.gouv lists API Adresse as an open Base Adresse Nationale API for address search, verification, geocoding, and reverse geocoding, with a published rate limit of 50 calls per second per IP.

For operators, this belongs in a compact slot because the receipt is narrow and immediately testable: an agent can turn 8 bd du port into candidate labels, scores, postcodes, city codes, and coordinates, then ask a human to confirm the right location before action.

Copy-Paste Agent Instruction

Use API Adresse to check this French address: "8 bd du port". Fetch the public search endpoint, return the top three candidate labels with score, postcode, citycode, and coordinates, then explain which ambiguity needs human confirmation before routing anything.

Creative Example

A support triage agent could normalize a messy French address before opening a field-service ticket, while flagging low-score matches or multiple cities with the same street name.

Test Caveat

The API is public, but geocoding is not identity proof. What to watch: whether agent workflows keep the score, city code, and coordinates visible instead of silently treating the first match as truth.

Primary Evidence

  • API.gouv API Adresse page: https://api.gouv.fr/les-api/base-adresse-nationale
    • Shows: API Adresse is listed as a fully open Base Adresse Nationale API with address search, verification, geocoding, reverse geocoding, and rate-limit information.
  • API.gouv autocompletion guide: https://api.gouv.fr/guides/autocompleter-adresse
    • Shows: the public guide for using API Adresse to autocomplete reliable, geolocated addresses.
  • API Adresse technical documentation: https://adresse.data.gouv.fr/outils/api-doc/adresse
    • Shows: the official technical documentation surface for the API.
  • Live API Adresse search query: https://api-adresse.data.gouv.fr/search/?q=8%20bd%20du%20port
    • Shows: a public endpoint response with candidate labels, scores, postcodes, city codes, and coordinates.

Evidence Limits

  • Address geocoding is not identity proof or legal eligibility proof.
  • The endpoint is public and rate-limited; agents should preserve scores and ambiguity instead of silently accepting the first match.

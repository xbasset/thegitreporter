# The Git Reporter Static Site

Generated static website for The Git Reporter.

This repository is intended to be served as plain static files, preferably with
GitHub Pages from the `main` branch root.

It is generated from:

```text
thegitreporter-production-workspace/.exports/static-site/
```

Do not add the private article pipeline, local repo caches, scanner output, raw
feedback exports, or recovered local source paths here.

Today Agent Pulse tracks 9 watched projects through public git motion, source receipts, and The Git Reporter coverage. Treat it as a reported signal board, not a definitive market ranking.

  • openclaw/openclaw is still the loudest source-code motion signal with 841 commits in seven days.

  • openclaw/openclaw cooled the most week-over-week, down 1,140 commits from the previous seven days.

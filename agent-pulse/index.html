Code motion is visible; community health is still the missing half.

Today Agent Pulse tracks 9 watched projects through public git motion, source receipts, and The Git Reporter coverage. Treat it as a reported signal board, not a definitive market ranking.

  • openclaw/openclaw is still the loudest source-code motion signal with 830 commits in seven days.
  • langchain-ai/langchain is the strongest baseline mover, 25.5 commits above its 30-day pace.

  • openai/codex has the deepest article trail on The Git Reporter, with 57 linked articles.
  • openclaw/openclaw cooled the most week-over-week, down 1,130 commits from the previous seven days.

V0.1 does not collect GitHub issue flow, PR review timing, Discord, X/Twitter, Reddit, Hacker News, or private backlog data.

Code motion is visible; community health is still the missing half.

Today Agent Pulse tracks 10 watched projects through public git motion, source receipts, and The Git Reporter coverage. Treat it as a reported signal board, not a definitive market ranking.

  • openclaw/openclaw is still the loudest source-code motion signal with 834 commits in seven days.
  • different-ai/openwork is the strongest baseline mover, 38.3 commits above its 30-day pace.
  • openclaw/openclaw leads the hardening proxy, but that proxy is built from commit-subject evidence, not release quality.
  • openai/codex has the deepest article trail on The Git Reporter, with 57 linked articles.
  • openclaw/openclaw cooled the most week-over-week, down 1,126 commits from the previous seven days.

V0.1 does not collect GitHub issue flow, PR review timing, Discord, X/Twitter, Reddit, Hacker News, or private backlog data.
